## What is VidPulse?

VidPulse is a backend service for YouTube video synchronization and classification. It helps you:

- **Sync videos** from YouTube channels you subscribe to
- **Enrich metadata** with detailed information
- **Classify videos** using hybrid (rules + LLM) classification
- **Provide APIs** and admin UI for management
