# Начало работы

Добро пожаловать в раздел "Начало работы"! Этот раздел содержит всю информацию, необходимую для начала использования VidPulse.

## Быстрый старт

Если вы новичок в VidPulse, мы рекомендуем следовать этим шагам:

1. **Руководство по быстрому старту** - Запустите VidPulse за 5 минут
2. **Руководство по установке** - Подробные инструкции по установке для разных сред
3. **Руководство по конфигурации** - Настройте VidPulse под свои нужды

## Руководства

### [Руководство по быстрому старту](./quick-start.md)

Краткое руководство для быстрого запуска VidPulse. Идеально подходит для ознакомления с сервисом.

### [Руководство по установке](./installation.md)

Подробные инструкции по установке с помощью Docker, ручной установке и развертыванию в production.

### [Руководство по конфигурации](./configuration.md)

Полное руководство по настройке VidPulse, включая переменные окружения, настройку базы данных и интеграции.
