## Краткое руководство по началу работы с VidPulse

VidPulse — это бэкенд-сервис для синхронизации и классификации видео с YouTube. Это руководство поможет вам запустить VidPulse за 5 минут.

### Предварительные требования

- Docker и Docker Compose
- YouTube Data API v3 ключ
- PostgreSQL (опционально, можно использовать встроенную SQLite)

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/your-org/vidpulse.git
cd vidpulse
```

### Шаг 2: Настройка окружения

Создайте файл `.env` на основе примера:

```bash
cp .env.example .env
```

Отредактируйте `.env`, указав ваш YouTube API ключ:

```env
YOUTUBE_API_KEY=your_youtube_api_key_here
```

### Шаг 3: Запуск с Docker Compose

```bash
docker-compose up -d
```

### Шаг 4: Проверка работы

Откройте браузер и перейдите по адресу `http://localhost:3000` для доступа к административному интерфейсу.

### Шаг 5: Добавление первого канала

Используйте административный интерфейс или API для добавления YouTube-канала.
