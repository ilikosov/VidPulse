## Ограничения и квоты

- **Ограничения скорости API**: 100 запросов в минуту на аутентифицированного пользователя
- **Квоты YouTube API**: Управляются автоматически с настраиваемыми лимитами
- **Ограничения классификации**: Настраиваются в зависимости от вашего провайдера OpenAI/LLM

## Коды ошибок

Распространенные коды ошибок и их значения:

- `VALIDATION_ERROR` - Неверные входные параметры
- `AUTHENTICATION_ERROR` - Ошибка аутентификации
- `AUTHORIZATION_ERROR` - Недостаточно прав
- `RESOURCE_NOT_FOUND` - Запрашиваемый ресурс не существует
- `RATE_LIMIT_EXCEEDED` - Слишком много запросов
- `INTERNAL_SERVER_ERROR` - Ошибка сервера

## Версионирование

API использует семантическое версионирование. Текущая версия: `v1`.

- **v1** - Текущая стабильная версия
- **v0** - Устаревшая, не поддерживается
