# 1. Общая цель проекта

Разработать backend-сервис с веб-админкой, который:

- получает список видео из подписанных YouTube-каналов через YouTube Data API
- синхронизирует новые видео в локальную базу данных
- извлекает и структурирует метаданные видео
- выполняет гибридную классификацию (rules + LLM)
- предоставляет API и веб-интерфейс для мониторинга и управления системой
