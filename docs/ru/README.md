## Что такое VidPulse?

VidPulse — это бэкенд-сервис для синхронизации и классификации видео с YouTube. Он помогает:

- **Синхронизировать видео** с YouTube-каналов, на которые вы подписаны
- **Обогащать метаданные** подробной информацией
- **Классифицировать видео** с помощью гибридной классификации (правила + LLM)
- **Предоставлять API** и административный интерфейс для управления
