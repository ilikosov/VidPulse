# 2. Основной функционал

## 2.1 Синхронизация видео

- получение списка каналов
- регулярная проверка новых видео
- поддержка backfill (история до заданной даты)
- инкрементальная синхронизация

Источник: YouTube Data API v3

---

## 2.2 Обогащение метаданных

Сохраняемые поля:

- title
- description
- publishedAt
- duration
- viewCount / likeCount / commentCount
- thumbnails
- channelId

---

## 2.3 Гибридная классификация

Режимы:

1. rule-based (быстро)
2. LLM-based extraction (сложные случаи)
3. hybrid (основной режим)

Пример результата:

```json
{
  "type": "fan_cam",
  "artist": "BLACKPINK",
  "group": "BLACKPINK",
  "song": "Pink Venom",
  "event": "World Tour",
  "location": "Seoul",
  "date": "2023-08-12"
}
```

---

## 2.4 Переклассификация

- поддержка изменения логики классификации
- массовая переклассификация
- версионирование классификатора
